# MVC_7-02-2026

โครงเริ่มต้นสำหรับข้อสอบ MVC (Rumor Tracking System)

## คำสั่งใช้งาน

```bash
npm install
npm run db:setup
npm run dev
```

เปิดที่ http://localhost:3000

เข้าสู่ระบบแบบ user/pass: http://localhost:3000/login

- schema: `sql/schema.sql`
- seed: `sql/seed.sql`
- db file: `data/rumour.db`
